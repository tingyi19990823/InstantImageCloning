## Dependency
### python 3.6.8
### opencv-python
### Pillow

## 操作方式:

### Step 1: 執行 main.py
### Step 2: 點選 open source image 開啟圖片
![image](https://github.com/tingyi19990823/InstantImageCloning/blob/master/ReadMePic/ReadMe1.jpg)

### Step 3: 用右鍵選擇要貼上的區塊，並按 "Crop" 然後按 "Done"
### 注: 中鍵可以取消選的前一個點
![image](https://github.com/tingyi19990823/InstantImageCloning/blob/master/ReadMePic/ReadMe2.jpg)
![image](https://github.com/tingyi19990823/InstantImageCloning/blob/master/ReadMePic/ReadMe3.jpg)

### Step 4: 點選 open target image 開啟圖片
### Step 5: 用右鍵選擇要貼上的中心座標，並按 "Wrap" 然後按 "Done"
![image](https://github.com/tingyi19990823/InstantImageCloning/blob/master/ReadMePic/ReadMe4.jpg)
![image](https://github.com/tingyi19990823/InstantImageCloning/blob/master/ReadMePic/ReadMe5.jpg)

### Step 6: 按 start blending 後就會得出結果，存檔按 save result
![image](https://github.com/tingyi19990823/InstantImageCloning/blob/master/ReadMePic/ReadMe6.jpg)
